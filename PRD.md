# 项目需求：新西兰 IT 求职市场分析工具 

## 1. 核心目标
抓取 IT 岗位数据并进行统计分析，生成每日就业市场分析海报 [cite: 24]。

## 2. 数据来源
- URL: https://me2link.com/job?selection=job-report&ppage=1

## 3. 功能模块 [cite: 26, 28]
- **数据抓取**：提取岗位名称、类别、技术关键词。
- **统计分析**：按类别整理岗位，统计技术词频。
- **海报生成**：尺寸固定为 1080×1440，展示岗位总数、分类分布、技术关键词。
- **图片导出**：使用 `html-to-image` 库支持 PNG 导出 [cite: 35, 36]。

## 4. 数据库 [cite: 23, 27]
- MySQL 数据库名：intern_nz_it_jobs
- 权限：基础增删改查
